Selectors & Specificity in CSS

Specificity Levels

Specificity Code Numeric Value Explanation
1.0.0.0 1000 Inline styles (e.g., style="color: red;") – The highest specificity, overrides all other styles unless !important is used elsewhere.
0.1.0.0 100 ID selectors (e.g., #myID) – Stronger than class and element selectors, but weaker than inline styles.
0.0.1.0 10 Class, attribute, and pseudo-class selectors (e.g., .button, [type="text"], :hover) – Applied when no ID or inline styles are present.
0.0.0.1 1 Element and pseudo-element selectors (e.g., div, h1, ::before) – The lowest specificity, overridden by class, ID, or inline styles.
0.0.0.0 0 Universal selector (*), combinators (>, +, ~), and pseudo-elements in some cases – No direct specificity, only applies when nothing stronger is present.

Basic Selectors

Selector Description Specificity Level
* Universal selector, selects all elements 0
element (e.g., div, h1, ::before) Type selector, selects all elements of the given type 1
.class Selects elements with a specific class 10
#id Selects an element with a specific ID 100
[attr] Selects elements with a specific attribute 10

Pseudo-elements

Selector Description Specificity Level
::before Inserts content before an element 1
::after Inserts content after an element 1
::first-letter Selects first letter of text 1
::selection Selects text when highlighted 1

Pseudo-classes

Selector Description Specificity Level
:hover Selects an element when hovered 10
:nth-child(n) Selects nth child of a parent 10
:first-child Selects first child of a parent 10
:focus Selects an element when focused 10

At-Rules

Selector Description Specificity Level
